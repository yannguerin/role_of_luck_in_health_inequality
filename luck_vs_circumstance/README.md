# Luck vs. Circumstance Model

## The Model (model.py)

This is the main model file that contains the neworder model used for both calibration and simulations.

## Distributions (dist.py)

Contains the code for generating the Effort and Circumstance distribution.

## Model Parameters (params.py)

Contains the code for loading the datasets of empirical data used in calibration.

## Dataset (datasets/)

Contains all the empirical datasets used by model and calibration.

## Calibration Interface (black_it_interface.py)

Contains the interface between the model and the calibration.
