# Results Folder

Warning: Changing the folder names or hierarchy, without changing the code, will results in errors.

## Calibration Results

Within this folder are the figures generated during the calibration process.

## Interactive Results

This folder stores the interactive results from the simulations, created to help the team better understand each simulation. Within this folder is a templates folder which contains Jinja templates used to generate the interactive results.

## Simulation Datasets

Within this folder are subfolders for each simulation containing the population, population over time, and cross sectional population datasets.
